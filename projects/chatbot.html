Chatbot Assistant

Tools Used: LangChain, OpenAI, Python, SQL, GCP

Project Summary:

This LLM-powered assistant enables supply chain teams to query logistics documents, SOPs, and catalogs using natural language. It significantly reduces the time spent searching for information in complex datasets and documentation.

Highlights:

  • Built with LangChain and OpenAI for robust document-based Q&A.
  • Indexed thousands of logistics SOPs and manuals for real-time querying.
  • Integrated with GCP services and deployed using Flask backend and React frontend.
  • Ranked in the top 1% most-used internal tools globally (100+ active users).
